QDC Encoder

Include dependency graph for QDC_Encoder.h:

This file contains declarations for an QDC Encoder.

This file contains class and function declarations for an encoder based on the QDC (Quadrature Decoder) peripheral.

Author

Jayson De La Vega, R&D Team

Date

2024-07-31

The QDC is a module within Teensy 4.x that decodes quadrature signals, eliminating the need to parse the signal in code. This is for hall-effect mmotor encoders (one of the more common variants for hobbyist applications), which typically emit quadrature signals. The Teensy’s QDC module also contains registers which are specifically suited for motor encoder applications.

Defines

MAX_NUM_ENCODER_INSTANCES

This macro defines the maximum number of encoder instances that the user may instantiate at a time.

The Teensy 4.x contains 4 QDC channels, so only 4 QDC Encoders may be instantiated at a time.
