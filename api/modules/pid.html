PID pidInit(float fKP, float fKI, float fKD, float fEpsilonInner, float fEpsilonOuter, float dInner, float dOuter)

Initializes a PID instance.

This function returns a PID struct which can be used to calculate a PID signal.

Parameters

  • fKP – proportional gain

  • fKI – integral gain

  • fKD – derivative gain

  • fEpsilonInner – uhhh, idk

  • fEpsilonOuter – uhhh, idk

  • dInner – uhhh, idk

  • douter – uhhh, idk

Returns

an instance of PID parameters

float pidCalculate(PID pid, float fSetPoint, float fProcessVariable, float dt)

Calculates PID signal.

This function calculates a float signal based on given PID parameters and a set point.

Parameters

  • pid – instance of PID structure

  • fSetPoint – set point of PID controller

  • fProcessVariable – sensor/feedback value

  • dt – delta time from last pid calculation

Returns

output value constrained from -255 to 255
